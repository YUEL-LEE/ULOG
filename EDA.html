EDA

EDA(Exploratory Data Analysis, 탐색적 데이터 분석)란 데이터 분석 방법 중 하나로, 데이터가 표현하고 있는 현상에 대해 이해하고, 그 데이터에 대한 잠재적인 문제와 아이디어를 발견할 수 있게 해요. EDA(Exploratory Data Analysis, 탐색적 데이터 분석)를 통해 데이터 수집 의사 결정, 추론한 가정에 대해 검토를 진행할 수 있고, 그에 맞도록 통계적 툴을 활용하거나 비교적 더 정확한 미래예측 모델을 만들 수 있어요. 데이터 분석은 데이터 탐색 - 선정 - 분석이라는 과정을 통해 이루어지는데, 데이터 탐색을 진행할 때 우리가 보아야 할 것은 행(가로)과 열(세로)의 수, 열의 Header(데이터 이름) 유무, Missing data(결측 데이터) 소실된 데이터, 원본 데이터의 형태 확인등이 있어요. 

            
                # 'pandas' library 불러오기
                import pandas as pd

                # import .csv(file)
                df = pd.read_csv(파일명)

                # columns = feature

                df.head()

그러면 데이터셋을 어떻게 불러올 수 있을까요? 우리는 일단 pandas library를 활용해 보아요. 여기서 library는 functions → modules(set) → package(folder) = library('set of packages')로 나타낼 수 있어요. library는 왜 써야할까요? 모든 기능을 직접 코딩하기 어렵고 시간이 오래 걸리기 때문이에요. import pandas as pd 라는 코드의 의미는 pandas module을 불러올 때 pandas 글자 수가 기니 앞으로는 pd로 줄여 불러오겠다는 말이에요. 우리는 df = pd.read_csv(파일명)를 통해 원하는 데이터셋을 가져올 수 있어요. df는 dataframe의 약자로, 우리는 변수를 보통 소문자/약자로 표현하기로 약속했어요. 그러니까 df = pd.read_csv(파일명)의 의미는 "'df'라는 변수에 pandas library를 통해 csv 확장자의 (파일명) 파일을 읽어와 줘!" 예요.

그리고 columns 즉, 열은 데이터셋의 특성을 나타내는 feature와 같은 뜻이라고 볼 수 있어요. 그래서 맨 위의 header는 종종 feature로 고정해 활용하기도 해요. df.head()는 상위 5개의 행을 출력하는 pandas library 예요. ()안에 숫자를 적어주면 그 숫자만큼 행을 출력할 수 있어요.
